<a name="module_telnet..TelnetClient+IsTelnet"></a>

#### telnetClient.IsTelnet(host, port) ⇒ <code>IsTelnetResponse</code>
IsTelnet checks if a host is running a Telnet server

**Kind**: instance method of [<code>TelnetClient</code>](#module_telnet..TelnetClient)  
**Returns**: <code>IsTelnetResponse</code> - - The response of the IsTelnet check.  
**Throws**:

- <code>error</code> - The error encountered during the IsTelnet check.


| Param | Type | Description |
| --- | --- | --- |
| host | <code>string</code> | The host to check for Telnet server. |
| port | <code>int</code> | The port to check for Telnet server. |

**Example**  
```js
let m = require('nuclei/telnet');
let c = m.TelnetClient();
let response = c.IsTelnet('localhost', 23);
```
