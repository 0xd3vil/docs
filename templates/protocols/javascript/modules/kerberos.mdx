#### kerberosClient.EnumerateUser(domain, controller, username) ⇒ <code>EnumerateUserResponse</code>
EnumerateUser returns true if the user exists in the domain. If the user is not found, false is returned. If the user is found, true is returned. Optionally, the AS-REP hash is also returned if discovered.

**Kind**: instance method of [<code>KerberosClient</code>](#module_kerberos..KerberosClient)  
**Returns**: <code>EnumerateUserResponse</code> - - The response of the enumeration.  
**Throws**:

- <code>error</code> - The error encountered during enumeration.


| Param | Type | Description |
| --- | --- | --- |
| domain | <code>string</code> | The domain to check. |
| controller | <code>string</code> | The controller to use. |
| username | <code>string</code> | The username to check. |

**Example**  
```js
let m = require('nuclei/kerberos');
let c = m.KerberosClient();
let response = c.EnumerateUser('domain', 'controller', 'username');
```
